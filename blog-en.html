Ready to Start Your Writing Journey

GitHub repository hasn't been configured yet. Here's a quick setup guide:

Step 1: Open blog.js

Step 2: Edit the GITHUB_CONFIG at the top:

const GITHUB_CONFIG = {
  owner: "your-GitHub-username",
  repo: "your-Obsidian-repo-name",
  branch: "main",
  contentPath: "blog"   // Which folder your articles are in
};

Step 3: Make sure the repository is Public and contains Markdown files.

Step 4: Refresh this page to see your articles!

💡 Article titles are automatically parsed from filenames (e.g. 2025-06-15-my-first-post.md) or from the first-level heading in your Markdown.
