准备开始你的写作之旅

还没有配置 GitHub 仓库信息。下面是一次性设置步骤：

第 1 步：打开 blog.js

第 2 步：修改顶部的 GITHUB_CONFIG：

const GITHUB_CONFIG = {
  owner: "你的-GitHub-用户名",
  repo: "你的-Obsidian-仓库名",
  branch: "main",
  contentPath: "blog"   // 你的文章在仓库中哪个文件夹
};

第 3 步：确认仓库是 Public（公开），里面有 Markdown 文件。

第 4 步：刷新此页面即可看到你的文章！

💡 文章的标题会从文件名（如 2025-06-15-我的第一篇.md）或 Markdown 中的一级标题自动解析。

😿

拉取文章时出了点小问题

未知错误

请检查：
① 仓库是否设置为 Public
② GITHUB_CONFIG 中的 owner/repo/branch/contentPath 是否正确
③ GitHub API 短时间内请求次数（60 次/小时），耐心等待或配置 Token
